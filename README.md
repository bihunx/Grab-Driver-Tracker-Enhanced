## Usage

1. **Open the Application**: Simply open `grabearn_v2.html` in any modern web browser
2. **Start Tracking**: 
   - Use the "Tracker" tab to add daily job counts
   - Set targets in the "Target" section for goal tracking
   - Record earnings and expenses in the "Earning" section
   - Manage additional finances in the "Financial" section
3. **Backup Regularly**: Use the backup feature to prevent data loss

## Data Storage

All data is stored locally in your browser's localStorage. No data is sent to external servers, ensuring complete privacy and security.
